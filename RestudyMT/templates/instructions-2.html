Part 1: Study phase

Instructions

In the first part of the experiment, you will be shown each pair of images for 4 seconds. The CUE will appear on the left side of the screen, and the TARGET will appear on the right.

INSTRUCTIONIMG[0]

While the images are on the screen, you should try to create a vivid mental image of the two objects interacting. Visual imagery is known to improve memory for associations. For example, if presented with the pair shown above you might imagine using the handle of the wagon to make sounds on the xylophone. This would help create a strong association betweent the CUE and TARGET that will come to mind more easily when you are tested later.
